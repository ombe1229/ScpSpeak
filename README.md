# ScpSpeak

SCP:SL 모든 SCP에게 939 보이스 능력을 부여하는 플러그인

Config 파일 세부화, 기능 추가 등은 디스코드 **ombe#3619**에게 유료로 구매 가능

## 기능

- 모든 SCP에게 939 보이스 능력을 부여

## Config

|    설정    | 변수 타입 | 기본값 | 설명                      |
| :--------: | :-------: | :----: | :------------------------ |
| is_enabled |  Boolean  |  true  | 플러그인을 키거나 끕니다. |
